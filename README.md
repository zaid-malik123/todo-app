# 📝 Todo App

A simple and responsive Todo App built using **React + Vite** with features like:

- ✅ Add & delete tasks
- 🗂️ Save tasks in **localStorage**
- ✅ Mark tasks as completed
- 📱 Fully responsive UI
- 🚀 Fast build & dev with Vite

---

## 🔧 Tech Stack

- ⚛️ React.js
- ⚡ Vite
- 💅 TailwindCSS
- 🔁 LocalStorage
- 🧼 React Hook Form

---



## 📂 Installation


# Install dependencies
npm install

# Start the local server
npm run dev
